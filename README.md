# fieldwork
set up temp log in selected rivers before the summer 2026, to be collected next year.
6 rivers were selected in the south-east region of Sweden. We discussed the set up: set 6 temp log per river,
3 in riffles and 3 in pools downstream of riffles section. The whole strectch should extend for several km 
(typically a river has one riffle section every 7-8). Standardize the method to minimiza variation within pool/riffle, 
i.e. 1) set loggers in the deepest point of pools (unclear how to mesure the wtaer depth, since it depends on
water level, which is fluctuating. Joakim suggest to measure depth from the bank, rather than the water level that day,
but it admit that it would be hard even for himself to judge). 2) set loggers downstream of riffles, in a point that
would not dry in case of drought but close enough to the riffle to mesure the outflow from it.
Measure: location, coordinates, type (riffle/pool), distance from riffle section, water velocity.
Use train break to anchor the logger, place it underneat them, set a rock on top if possible, mark with blue spray
the point on land and take GPS coordinates. Maybe add a note "don't remove" and contact number.

These sites are electrofished, however we will have data only for the riffles, not for the pool.

Tributaries should not be a problem as these are first order rivers, and the tributaries are ditches.
